== README

## Registrerings-applikation

## Ruby version
* Ruby 2.1.5
* Rails 4.2.5

## För att köra appen
```bash
git clone https://github.com/ad222kr/1dv450-registration.git 
bundle install --without production ``` 
rake db:setup
rails s 
```  
Applikationen hittas nu på localhost:3000
